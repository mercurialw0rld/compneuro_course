# Google Colab Notebooks

This directory contains all the Google Colab notebooks from the course, organized by week.

## Organization

- Each week has its own subdirectory (week1, week2, etc.)
- Notebooks should be named descriptively, e.g., `week1_neural_encoding.ipynb`

## Adding Notebooks

### Method 1: Direct Download from Colab
1. Open your notebook in Google Colab
2. Go to `File → Download → Download .ipynb`
3. Save to the appropriate week folder in this directory

### Method 2: Save to GitHub from Colab
1. In Google Colab: `File → Save a copy in GitHub`
2. Select repository: `mercurialw0rld/compneuro_course`
3. File path: `notebooks/weekN/your_notebook_name.ipynb`
4. Add a commit message and save

## Naming Convention

Use descriptive names for your notebooks:
- `week1_introduction.ipynb`
- `week2_hodgkin_huxley.ipynb`
- `week3_spike_trains.ipynb`
- etc.
