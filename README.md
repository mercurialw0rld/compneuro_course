## How to Use This Repository

### Saving Google Colab Notebooks

1. **From Google Colab**: 
   - File → Download → Download .ipynb
   - Save the notebook to the appropriate week folder

2. **Alternatively, sync with GitHub directly from Colab**:
   - File → Save a copy in GitHub
   - Select this repository
   - Choose the appropriate folder path

### Running the Notebooks

You can run these notebooks in several ways:

1. **Google Colab** (Recommended):
   - Upload the notebook to [Google Colab](https://colab.research.google.com/)
   - Or open directly from GitHub using Colab

2. **Local Jupyter**:
   ```bash
   pip install jupyter
   jupyter notebook
   ```
